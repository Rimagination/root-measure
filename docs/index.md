# Root Measure Tutorial

This is the GitHub Pages tutorial home for `root-measure`.

This site is intentionally separate from the plugin `README`:

- the plugin `README` answers "what this is, how to ask Codex to use it, and where to find results"
- GitHub Pages holds fuller tutorials, parameter guidance, reproduction notes, and validation records

## Who should read this

- If you mainly use the plugin through Codex, start with the repository README:
  [Rimagination/root-measure](https://github.com/Rimagination/root-measure)
- If you want to understand parameters, modes, and reproduction boundaries, start here
- If you want to audit a specific exact reproduction workflow, go directly to `Reproducibility`

## Navigation

- [Getting Started](./getting-started/)
- [Parameters and Modes](./parameters-and-modes/)
- [Reproducibility](./reproducibility/)

## Current scope

This tutorial site currently focuses on:

- how ordinary users can complete a first analysis through Codex
- how to choose between `broken roots` and `whole root / crown`
- what the most important exposed parameters change
- where validated public-data reproduction paths do and do not generalize
